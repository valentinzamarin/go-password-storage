# Go Password Storage

This is a small application written in Golang that I created for personal use.

Stack: 
- Go 1.23+
- Fyne (GUI)
- GORM (ORM)
- SQLite
- Argon2id (key derivation)
- NaCl/secretbox (encryption)

Architecture
- Domain-Driven Design (DDD)
- CQRS pattern
- Clean Architecture
